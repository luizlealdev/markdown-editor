# Markdown Example

## Formatting Text

*Italic text* can be created by surrounding the text with asterisks or underscores.

**Bold text** can be created by surrounding the text with double asterisks or double underscores.

## Code Blocks

Inline code can be included with backticks, like this: `inline code`.

For a block of code, use triple backticks:

```python
def hello_world():
    print("Hello, world!")
```
